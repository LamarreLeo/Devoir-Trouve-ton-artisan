# Trouve Ton Artisan

## Description du projet

"Trouve Ton Artisan" est une plateforme qui permet aux utilisateurs de trouver facilement des artisans qualifiés dans la région Auvergne-Rhône-Alpes, de consulter leurs profils, leurs notes, et de les contacter via un formulaire de contact.

## Fonctionnalités principales

-   Recherche d'artisans par catégorie ou par nom
-   Consulter les notes des artisans
-   Contacter les artisans via un formulaire de contact
-   Profils détaillés des artisans

## Backend

Le backend est développé en Node.js avec Express.js

## Frontend

Le frontend est développé en React.js
